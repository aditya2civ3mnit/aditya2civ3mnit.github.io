# SMC Trade Journal - 1-Page Investor Brief

## Elevator Pitch
SMC Trade Journal is a workflow-first trading performance platform that transforms discretionary trading into a repeatable, auditable process from pre-trade planning through post-trade review.

## Problem
Retail and independent traders lose consistency because their process is fragmented:
- Strategy logic is not operationalized
- Entries are made without checklist discipline
- Trade evidence is scattered
- Review is subjective and rarely data-backed

Result: recurring execution mistakes, low accountability, and slow improvement.

## Solution
SMC Trade Journal creates a closed performance loop:
1. Build strategy logic in a Setup Canvas
2. Execute through dynamic, setup-driven checklists
3. Capture chart evidence per setup segment
4. Replay and review with structured post-trade analysis
5. Track consistency with monthly backend analytics

## Product Differentiators
- Setup-driven JSON architecture: strategy changes without code rewrites
- End-to-end workflow in one tool: plan, execute, review, measure
- Segment-level media proof and replay: visual accountability
- Backend-first analytics: trustworthy reporting over UI-only calculations
- Trading-process focus: built for execution quality, not only journaling text

## Core Features
- Authentication and user-scoped data security
- Setup Canvas with nested sub-items and If/Else branch logic
- Dynamic pre-trade and post-trade checklist rendering
- Trade state/outcome tracking with review notes and P/L fields
- Media uploads by setup segment (image/video + paste workflow)
- Slide-based replay across pre-trade and post-trade segments
- Trade history dashboard with filtering and archive/restore lifecycle
- Monthly green/red/flat day analytics with weekly aggregates
- Floating risk and lot-size calculator for execution planning

## Target Users
- Serious retail traders
- Prop-firm aspirants
- Trading mentors and coaching groups
- Small trading teams that need process QA

## Business Value to User
- Reduces impulsive entries through pre-trade gating
- Improves strategy adherence and execution discipline
- Speeds up learning via replayable evidence and structured review
- Creates measurable process metrics linked to outcomes

## Monetization Paths
- Subscription tiers (Solo, Pro, Team)
- Mentor/coach workspaces
- Premium analytics and benchmarking modules
- Broker/data API integrations as paid add-ons

## Expansion Opportunities
- Live chart-data replay integrations
- AI-assisted trade review and mistake tagging
- Strategy performance attribution by setup branch
- Team collaboration and reviewer permissions

## Why Now
The rise of discretionary trading communities and prop challenges increases demand for process accountability tools. Most traders still operate with notes and screenshots; SMC Trade Journal delivers a structured operating system for consistent execution.
